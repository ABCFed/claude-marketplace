# 分支命名规则

## 功能分支 (feature)

格式：`feature/<功能名>-<TAPD-ID>`

示例：
- `feature/stat-commission-1023456`
- `feature/user-profile-1023457`

## Hotfix 分支 (hotfix)

格式：`hotfix/<问题描述>-<TAPD-ID>`

示例：
- `hotfix/cant-charge-1023458`
- `hotfix/login-error-1023459`

## Hotfix-g 分支 (hotfix-g)

格式：`hotfix-g/<问题描述>-<TAPD-ID>`

示例：
- `hotfix-g/payment-fail-1023460`
- `hotfix-g/data-sync-1023461`

## 命名建议

- 使用英文小写
- 用连字符 (-) 连接多个单词
- 包含 TAPD ID 方便追溯
- 简洁描述功能或问题
- 避免使用中文或特殊字符
